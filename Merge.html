Merge

O comando "merge" no Git é usado para combinar as mudanças de diferentes branches (ramificações) em uma única branch. Isso é especialmente útil quando você está trabalhando em uma feature ou correção de bug em uma branch separada e deseja integrar essas mudanças de volta à branch principal (geralmente a "main" ou "master"). O comando merge analisa as diferenças entre as branches e aplica as alterações, criando um novo commit de merge que une o histórico das duas branches.
