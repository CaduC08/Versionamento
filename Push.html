Push

O comando "push" no Git é usado para enviar (ou "empurrar") suas alterações locais para um repositório remoto, como o GitHub. Isso torna suas mudanças acessíveis a outros colaboradores e permite que o código seja compartilhado e integrado com o trabalho de outros desenvolvedores. O comando push geralmente é usado após um commit, garantindo que as alterações estejam salvas localmente antes de serem enviadas para o repositório remoto.
