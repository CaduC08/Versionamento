Fetch

O comando "fetch" no Git é usado para baixar atualizações de um repositório remoto para o seu repositório local, sem mesclar essas mudanças automaticamente. Isso permite que você veja as alterações feitas por outros colaboradores antes de integrá-las ao seu trabalho atual. O comando fetch é útil para manter seu repositório local atualizado com o estado mais recente do repositório remoto, facilitando a colaboração e o gerenciamento de versões.
